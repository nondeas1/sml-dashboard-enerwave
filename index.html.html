    Glossary
    Metric definitions + Power BI integration guide
    Buzz / Distinct Mentions
    COUNT DISTINCT of table[id] — each unique content piece = 1 Buzz unit. Engagement type rows (Likes/Shares etc.) share the same id, so de-duplication is essential before counting.
    Share of Voice (SOV)
    Brand Buzz ÷ Total Sector Buzz × 100. DAX: DIVIDE(CALCULATE([Buzz],Queries[query_name]="Enerwave"),[Total Buzz])*100
    Earned Media Value (EMV)
    SUM of table[emv] after de-duplicating by id. Represents the estimated cost of equivalent paid media placement.
    Impact Score
    AVG of table[impact] — proprietary score (0–1) per mention reflecting source authority and reach potential. DAX: AVERAGEX(DISTINCT(table[id]),RELATED(table[impact]))
    BMU Category
    Dimension BMUI[bmu_category]: Brand · Media · UGC · Nano-influencers (<10K) · Micro (10K–100K) · Mid-tier (100K–500K) · Macro (500K–1M) · Mega (>1M followers).
    Emotion Detection
    Dimension Emotions[emotion]: Happiness · Anger · Sadness · Fear · Neutral. Deeper NLP layer complementing Sentiment — identifies the why behind tone.
    Power BI Integration
    Replace window.SML_DATA at the top of this file with a DAX measure generating a JSON string from your live aggregates. The HTML Content visual passes the measure value to this variable, making all charts live. Dimensions: Calendar · Queries · Channels · Sentiment · Emotions · BMUI · BMUI_Influencers · TopicSlicer.
